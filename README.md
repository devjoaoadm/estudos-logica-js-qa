# Estudos de Lógica com JavaScript para QA 🚀

Este repositório armazena meus estudos fundamentais de lógica de programação utilizando JavaScript, com foco total em preparar a base para automação de testes (QA).

## 📂 O que tem aqui?

Atualmente, o projeto conta com a pasta `conceitos-js`, que aborda:
* **Variáveis e Tipos de Dados:** Entendendo como manipular informações.
* **Estruturas de Decisão:** Lógica de `if/else` para fluxos de teste.
* **Funções:** Criação de blocos de código reutilizáveis (essencial para Page Objects).
* **Estruturas de Repetição:** Manipulação de listas e arrays.

## 🛠️ Tecnologias
* JavaScript (Node.js)
* Git & GitHub
